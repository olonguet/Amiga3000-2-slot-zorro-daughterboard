# Amiga 3000 2 slot Zorro Daughterboard

<img width="1158" height="501" alt="1000 - Copy (2)" src="https://github.com/user-attachments/assets/7d80787d-fb05-44e8-8173-44d9a2d1791f" />

A scaled down version of the Zorro daughter board for the Amiga 3000 / 4000.
Using Ians Bremmer ( https://github.com/iansbremner/Amiga-3000-Daughterboard ) repro of the standard Amiga3000 Zorro daughterboard I needed to make a smaller, shorter version of it to fit in a Checkmate 1500 case (the original being too high).
I decided to make it a minimalist 2 slot version needing the bare mimimum for the A3000 to boot. 

<img width="1711" height="1016" alt="1000 - Copy (2) - Copy" src="https://github.com/user-attachments/assets/2560f24f-4f92-4298-ae91-5cd5cf8b9df8" />

This is part of a bigger project involving a 3D printable kit to fit an Amiga3000 into a Checkmate 1500 case: https://www.thingiverse.com/thing:7326858

https://www.youtube.com/watch?v=aT_P7sj8tz4

NOTE: If you want a strict mimimum version, populating RP605 should be all that's needed to power your A3000. RP605 It's a dual terminator resistor pack so pin 1 is a pull down to ground at 330 and pin 8 is a pull up at 220 to +5v
