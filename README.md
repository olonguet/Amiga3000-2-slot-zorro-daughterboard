A scaled down version of the Zorro daughter board for the Amiga 3000 / 4000
Using Ians Bremmer repro of the standard Amiga3000 Zorro daughterboard I needed to make a smaller, shorter version of it to fit in a Checkmate 1500 case (the original being too high).
I decided to make it a minimalist 2 slot version needing the bare mimimum for the A3000 to boot. 

NOTE: If you want a strict mimimum version, populating RP605 ahd RP608 should be all that's needed to power your A3000
